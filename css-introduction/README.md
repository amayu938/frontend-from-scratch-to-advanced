# CSS Introduction

This folder contains the solution for the CSS introduction challenge.

## Files Included

- `index.html` — HTML structure for a simple developer profile card
- `style.css` — CSS styling that demonstrates basic box model and layout techniques
- `Screenshot 2026-06-17 210005.png` — screenshot showing the rendered design

## What This Project Shows

- How to apply CSS to a card layout
- Basic use of margins, padding, and borders
- Applying font and color styling
- Clean presentation of a simple personal profile section

## How to View

Open `index.html` in a browser to see the card layout and styling.
