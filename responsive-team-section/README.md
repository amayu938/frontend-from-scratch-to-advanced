# Responsive Team Section

A responsive team section built for Challenge 09.

## What is included

- 3 team member cards
- Name, role, and short description for each member
- Desktop layout using Flexbox with cards in one row
- Mobile layout using a media query with cards stacked in a column
- Required CSS features: `display: flex`, `gap`, `flex-direction`, `padding`, `border-radius`, and a media query

## Files

- `index.html` — the team section HTML structure
- `style.css` — responsive styling for desktop and mobile
- `screenshot.png` — placeholder screenshot file
