# CSS Box Model

This folder contains the solution for the CSS box model challenge.

## Files Included

- `index.html` — HTML structure for a developer profile card
- `style.css` — CSS styling that demonstrates box model concepts
- `Screenshot 2026-06-17 210140.png` — screenshot showing the rendered design

## What This Project Shows

- Use of padding and margin to create spacing
- A styled card with border, background, and shadow
- Centering content using CSS layout techniques
- Clean typography and visual hierarchy for a profile card

## How to View

Open `index.html` in a browser to see the box model card design.
